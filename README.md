# ResourceTool Wiki

ResourceTool 官方文档站点，基于 [VuePress 2](https://v2.vuepress.vuejs.org/zh/) + [vuepress-theme-plume](https://theme-plume.vuejs.press/) 构建。

## 在线访问

部署在 GitHub Pages 上，访问地址：[https://shutu777.github.io/resourcetool-wiki/](https://shutu777.github.io/resourcetool-wiki/)

## 本地开发

```bash
# 安装依赖
npm install

# 本地预览
npm run dev

# 构建
npm run build
```

## 部署方式

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。

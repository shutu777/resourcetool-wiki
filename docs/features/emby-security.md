# 安全策略

对应前端侧边栏 **开服 → 安全策略**，配置 Emby Bot 的安全和注册策略，按 Emby 实例独立管理。

## 页面结构

顶部标题栏 + Emby 实例选择下拉。选择实例后显示该实例的安全策略配置表单，底部有「保存」按钮。

::: details 定时任务
| 配置项 | 说明 |
|--------|------|
| **密码重置 Cron** | 定时批量重置用户密码的 Cron 表达式 |
| **过期扫描 Cron** | 定时扫描过期用户并自动禁用的 Cron 表达式 |
| **用户同步 Cron** | 定时将 RT 用户状态同步到 Emby 的 Cron 表达式 |
:::

## 注册策略

| 配置项 | 说明 |
|--------|------|
| **开放注册** | 开关。开启后允许用户通过 Bot 自助注册 |
| **需要邀请码** | 开关。开启后注册时必须提供有效邀请码 |
| **默认时长（天）** | 新注册用户的默认有效天数。填 0 = 永久 |
| **默认网盘模式** | 新注册用户的默认模式：共享 / 自备 / 两者皆可 |

::: caution 开放注册后，任何人均可通过 Bot 自助注册。建议同时开启「需要邀请码」来控制注册范围。
:::

## 设备限制

| 配置项 | 说明 |
|--------|------|
| **默认设备上限** | 新注册用户默认的最大同时播放设备数 |

::: info 设备数超出上限时，用户将无法在新设备上播放，需在其他设备上停止播放或联系管理员调整上限。
:::

## 防暴力破解

::: warning 重要
分别为三种场景配置防刷保护参数，超过限制后该 IP/用户将被临时锁定。
:::

### 登录保护

| 配置项 | 说明 |
|--------|------|
| **最大尝试次数** | 在时间窗口内允许的最大登录尝试次数 |
| **时间窗口（秒）** | 计数的时间窗口长度 |
| **锁定时长（秒）** | 超过限制后的锁定时间 |

### 注册保护

| 配置项 | 说明 |
|--------|------|
| **最大尝试次数** | 在时间窗口内允许的最大注册尝试次数 |
| **时间窗口（秒）** | 计数的时间窗口长度 |
| **锁定时长（秒）** | 超过限制后的锁定时间 |

### 密码重置保护

| 配置项 | 说明 |
|--------|------|
| **最大尝试次数** | 在时间窗口内允许的最大密码重置尝试次数 |
| **时间窗口（秒）** | 计数的时间窗口长度 |
| **锁定时长（秒）** | 超过限制后的锁定时间 |

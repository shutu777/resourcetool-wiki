# 缓存管理

对应前端侧边栏 **概览 → 缓存管理**，用于查看和管理 Emby 302 直链缓存，页面每 10 秒自动刷新。

## 页面概览

顶部标题栏显示页面说明和「清空全部缓存」按钮。下方以列表形式展示当前所有活跃的直链缓存条目，右上角显示条目总数。

## 缓存条目

每条缓存记录以带左侧蓝色边框的卡片展示，包含以下信息：

| 字段 | 说明 |
|------|------|
| **115 配置名** | 该直链使用的 115 账号配置名称 |
| **缓存时间** | 直链被缓存的时间 |
| **剩余倒计时** | 缓存到期前的剩余时间，格式为「X分XX秒」，实时倒计时 |
| **播放文件** | 正在播放的文件名或文件路径 |
| **115 直链** | 实际的 115 CDN 下载链接（点击可复制） |
| **Emby 用户** | 触发该缓存的 Emby 用户名 |
| **Host** | 请求来源的 Host |
| **UA** | 客户端 User-Agent |

如果缓存条目没有用户信息（无 Emby 用户、Host、UA），则会显示「预缓存」标签，表示该缓存由预缓存机制自动生成。

## 操作

- **复制直链** — 点击直链区域即可复制 115 302 链接到剪贴板
- **删除单条** — 每条记录右侧的 × 按钮可删除该缓存条目
- **清空全部** — 顶部红色按钮一键清空所有缓存

## 自动过期

::: important 自动过期
缓存条目到期后会自动从列表中消失。倒计时归零显示「已过期」后，条目不再展示。缓存过期时间由 Emby 302 配置中的「URL 缓存过期时间」决定。
:::

# CloudDrive2 助手

对应前端侧边栏 **插件库 → CloudDrive2 助手**，配置 CloudDrive2（CD2）的连接信息和挂载路径映射，为 Emby 302 直链播放提供 gRPC 加速。

## 功能概述

::: important 核心用途
在 Emby 302 的**路径替换**播放模式中，系统需要将 Emby 媒体路径转换为 115 网盘的文件 Pickcode。CloudDrive2 助手通过 gRPC 协议直连 CD2 服务，实现毫秒级的路径查询，大幅提升播放响应速度。
:::

## 页面结构

进入插件后，页面分为两个区域：**连接配置** 和 **挂载路径映射**。

### 连接配置

| 配置项 | 说明 |
|--------|------|
| **CD2 地址** | CloudDrive2 服务地址（如 `http://192.168.1.100:19798`） |
| **用户名** | CD2 登录用户名 |
| **密码** | CD2 登录密码 |
| **测试连接** | 点击按钮验证连接是否成功 |

::: tip 连接测试
配置完成后务必点击「测试连接」确认可达。连接失败时 302 播放会回退到其他模式，响应速度会下降。
:::

### 挂载路径映射

配置 Emby 媒体库路径与 CD2 挂载路径之间的对应关系：

| 配置项 | 说明 |
|--------|------|
| **Emby 路径前缀** | Emby 媒体库中的路径前缀（如 `/mnt/media/`） |
| **CD2 挂载路径** | 对应的 CD2 115 网盘挂载路径（如 `/115/media/`） |

::: details 路径映射示例
假设你的 Emby 媒体库路径为 `/mnt/media/movies/xxx.mkv`，CD2 将 115 网盘挂载到 `/115/media/`，则：

| 配置项 | 值 |
|--------|-----|
| **Emby 路径前缀** | `/mnt/media/` |
| **CD2 挂载路径** | `/115/media/` |

系统会将 `/mnt/media/movies/xxx.mkv` 转换为 `/115/media/movies/xxx.mkv`，然后通过 CD2 gRPC 查询该路径对应的 115 Pickcode。
:::

支持添加多组映射，适配多个媒体库路径。每组映射可独立编辑或删除。

## 使用方法

::::: steps

1. **安装 CloudDrive2**

   确保 CD2 已安装并挂载了 115 网盘

2. **配置连接**

   填写 CD2 的地址、用户名、密码，点击测试连接

3. **配置路径映射**

   添加 Emby 媒体路径与 CD2 挂载路径的对应关系

4. **在 Emby 助手中启用**

   前往 [Emby 助手](emby302.md) 配置路径替换模式，系统将自动使用 CD2 gRPC 查询

:::::

## 注意事项

::: warning CD2 服务可用性
CD2 服务需保持运行状态。如果 CD2 服务停止或挂载丢失，路径替换模式将无法正常工作，302 播放会尝试回退到 Pickcode 模式。
:::

::: caution 网络延迟
建议 CD2 与 ResourceTool 部署在同一内网环境中，避免因网络延迟影响播放响应速度。
:::

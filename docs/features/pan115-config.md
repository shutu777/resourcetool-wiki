# 115 助手

对应前端侧边栏 **概览 → 115 助手**，用于管理所有 115 网盘账号配置。页面以表格形式展示所有配置，支持搜索和按来源筛选。

## 页面结构

顶部标题栏包含「内置账号」和「自备账号」两个新增按钮。下方为搜索框（按名称搜索）和来源筛选下拉（全部 / 内置 / 自备）。

### 配置列表表格

| 列名 | 说明 |
|------|------|
| **名称** | 配置名称（如 `115-主号`） |
| **Cookie** | 脱敏显示的 Cookie（前 6 + 后 6 位） |
| **来源** | 「内置」（琥珀色）或「自备」（蓝色）标签 |
| **副 Bot** | 绑定副 Bot 时显示运行状态（运行中 / 已停止） |
| **清理** | 开启定时清理时显示 Cron 表达式 |
| **同播复制** | 开启时显示「开启」标签 |
| **操作** | 测试 Cookie / 编辑 / 删除 |

## 两种账号类型

::: card title="内置账号" icon="material-symbols:admin-panel-settings"

### 内置账号

由管理员配置的 115 账号，用于系统内部的转存、离线、Emby 302 播放等功能。支持完整配置：

| 配置项 | 说明 |
|--------|------|
| **名称** | 配置标识名 |
| **Cookie** | 115 网盘 Cookie，右侧「测试」按钮可验证有效性 |

**目录配置（仅内置）：**

| 配置项 | 说明 |
|--------|------|
| **转存目录** | 默认转存目标目录，通过目录选择器浏览 115 网盘目录树选择 |
| **离线目录** | 离线下载目标目录，同样使用目录选择器 |

::: details Telegram 副 Bot（仅内置）
每个内置 115 账号可绑定一个独立的 Telegram 副 Bot：

| 配置项 | 说明 |
|--------|------|
| **副 Bot Token** | 绑定专属 Bot 后，转存通知和命令将通过副 Bot 收发 |
| **白名单** | 只有这些用户可以使用副 Bot 命令和转存，留空则不限制 |
| **Chat ID** | 配置副 Bot Token 时必填，副 Bot 将向这些聊天发送通知 |

副 Bot 功能与主 Bot 类似，包含搜索影视、HDHive 用户管理、监控转发/转存查看、日志和版本信息等菜单。
:::

::: details 302 设置（仅内置）
| 配置项 | 说明 |
|--------|------|
| **同播复制** | 开关。开启后当多个用户同时播放同一文件时，自动秒传到该账号避免 115 并发限制 |
| **临时目录** | 复制小号秒传时的默认存储目录，默认 `/ResourceTool` |
:::

::: details 定时清理（仅内置）
| 配置项 | 说明 |
|--------|------|
| **启用定时清理** | 开关 |
| **清理 Cron** | Cron 表达式，依次执行：清空临时目录 → 清空回收站。默认 `0 1 * * *` |
| **回收站密码** | 清空回收站的安全密码 |
| **立即清理** | 手动触发一次清理（保存后可用） |
:::

### 自备账号

由 Emby 用户自行提供的 115 账号，仅需配置基础信息：

| 配置项 | 说明 |
|--------|------|
| **名称** | 配置标识名 |
| **Cookie** | 115 网盘 Cookie |
| **回收站密码** | 自备网盘清理回收站时需要的安全密码 |

## 目录选择器

::: tip 目录选择器
编辑配置时点击转存 / 离线目录右侧的文件夹按钮，会打开目录选择器弹窗。顶部面包屑导航可快速返回上级目录，每个子目录显示「选择」和「进入」两个按钮，使用当前配置的 Cookie 浏览 115 网盘。
:::

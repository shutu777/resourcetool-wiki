# 邀请码

对应前端侧边栏 **开服 → 邀请码**，管理 Emby 用户的邀请码（激活码），支持单个生成和批量生成。

::: important 核心用途
邀请码是控制 Emby 注册范围的核心手段。在 [开服配置](emby-security.md) 中开启「需要邀请码」后，用户注册时必须提供有效邀请码。邀请码决定了用户的**时长档位**、**网盘模式**和**所属实例**。
:::

## 页面结构

顶部标题栏显示邀请码统计（总数 / 未使用），右侧有实例筛选下拉（多实例时）、「生成单个」和「批量生成」按钮。下方为邀请码列表，支持分页和多选操作。

## 邀请码列表

| 列名 | 说明 |
|------|------|
| **邀请码** | 随机生成的字符串，可选中复制 |
| **实例** | 邀请码所属的 Emby 实例（多实例时显示） |
| **天数** | 对应的有效天数，白名单码显示 <Badge type="outline" text="白名单" /> |
| **模式** | 仅自备 / 仅共享 / 共享+自备(默认共享) / 共享+自备(默认自备) |
| **状态** | <Badge type="tip" text="未使用" /> / <Badge type="info" text="已使用" /> |
| **使用者** | 已使用时显示 Emby 用户名和使用日期 |
| **创建时间** | 邀请码创建时间 |

## 生成邀请码

### 生成单个

点击「生成单个」按钮，在弹窗中选择配置后立即生成一个邀请码并自动复制到剪贴板，适合即时发给单个用户。

### 批量生成

点击「批量生成」按钮，在弹窗中配置后一次性生成多个邀请码，生成后可一键复制全部。

### 生成配置项

| 配置项 | 说明 |
|--------|------|
| **数量** | 批量生成时的数量（1~100），单个生成时固定为 1 |
| **时长档位** | 天卡(1天) / 周卡(7天) / 月卡(30天) / 年卡(365天) / 白名单(永久) |
| **网盘模式** | 决定用户注册后的网盘使用方式 |
| **所属实例** | 多实例时选择邀请码绑定到哪个 Emby 实例，用户兑换时会校验实例匹配 |

### 网盘模式说明

| 模式 | 说明 |
|------|------|
| **仅自备** | 用户必须提供自己的 115 Cookie 才能播放 |
| **仅共享** | 用户使用管理员配置的共享 115 账号播放 |
| **共享+自备(默认共享)** | 用户默认使用共享模式，可自行切换到自备 |
| **共享+自备(默认自备)** | 用户默认使用自备模式，可自行切换到共享 |

::: tip 网盘模式默认值
生成弹窗中的网盘模式默认值来自 [开服配置](emby-security.md) 中的「默认网盘模式」设置。
:::

## 典型使用流程

::::: steps

1. **配置注册策略**

   在 [开服配置](emby-security.md) 中开启「开放注册」和「需要邀请码」

2. **生成邀请码**

   根据需要选择时长档位和网盘模式，单个生成或批量生成

3. **分发给用户**

   将邀请码发送给用户，用户在注册页面或 Bot 中使用

4. **用户兑换**

   用户注册时输入邀请码，系统校验邀请码的有效性、实例匹配和使用状态

5. **管理追踪**

   在列表中查看每个邀请码的使用情况，按实例筛选管理

:::::

## 批量操作

- **全选/取消** — 表格顶部的复选框可全选当前页
- **批量删除** — 选中后点击工具栏「删除」按钮，未使用和已使用的邀请码均可删除

::: warning 邀请码实例绑定
多实例环境下，邀请码绑定了特定实例。用户在注册时选择的实例必须与邀请码的所属实例一致，否则兑换会失败。
:::

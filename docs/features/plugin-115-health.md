# 115 健康检查

对应前端侧边栏 **插件库 → 115 健康检查**，用于批量检测所有 115 网盘账号的 Cookie 有效性，快速定位失效配置。

## 功能概述

::: important 核心用途
115 网盘 Cookie 会定期过期，过期后转存、离线、302 播放等功能均会失效。本插件可一键检测所有配置的 Cookie 状态，避免因 Cookie 失效导致服务中断。
:::

## 页面结构

进入插件后，页面展示所有已配置的 115 账号列表，每个账号显示：

| 字段 | 说明 |
|------|------|
| **配置名称** | 115 账号的配置标识名 |
| **Cookie 状态** | <Badge type="tip" text="正常" /> 或 <Badge type="danger" text="失效" /> |
| **来源** | 内置账号 / 自备账号 |
| **检测时间** | 最近一次检测的时间 |

## 使用方法

::::: steps

1. **进入插件**

   在前端侧边栏点击 **插件库 → 115 健康检查**

2. **执行检测**

   点击页面顶部的「一键检测」按钮，系统将逐个调用 115 API 验证每个账号的 Cookie

3. **查看结果**

   检测完成后，每个账号旁显示状态标签：
   - <Badge type="tip" text="正常" /> — Cookie 有效，无需操作
   - <Badge type="danger" text="失效" /> — Cookie 已过期，需要更新

4. **更新失效 Cookie**

   对于失效账号，前往 [115 助手](pan115-config.md) 编辑对应配置，更新 Cookie

:::::

## 使用建议

::: tip 定期检查
建议每周执行一次健康检查，或在发现转存/播放异常时优先检查 Cookie 状态。
:::

::: warning 检测频率
频繁调用 115 API 可能触发风控，建议不要在短时间内反复执行检测。
:::

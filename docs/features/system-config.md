# 系统配置

对应前端侧边栏 **系统 → 系统配置**，管理 ResourceTool 的全局设置。页面以多个配置区块纵向排列，底部有统一的「保存」按钮。

## 授权信息

页面顶部展示当前授权状态：

| 字段 | 说明 |
|------|------|
| **授权状态** | 已授权（绿色）/ 未授权（红色） |
| **授权码** | 密码框显示，可切换明文 |
| **设备 ID** | 当前绑定的设备标识 |
| **到期时间** | 授权到期日期 |
| **授权功能** | 已授权的功能模块列表 |

## 网络代理

| 配置项 | 说明 |
|--------|------|
| **代理地址** | HTTP/HTTPS/SOCKS5 代理地址（如 `http://127.0.0.1:7890`） |

::: info 智能分流
系统自动按域名进行智能分流：
- **走代理** — Telegram API、TMDB API、Telegraph、HDHive API
- **直连** — 115 网盘 API（国内服务）
:::

## TMDB 配置

| 配置项 | 说明 |
|--------|------|
| **TMDB API Key** | 用于影视搜索的元数据获取（海报、简介、评分等） |

## 配置导入导出

:::: card-grid
::: card title="导出配置" icon="material-symbols:upload-file"
将当前所有配置打包为 JSON 文件下载
:::
::: card title="导入配置" icon="material-symbols:download"
上传 JSON 配置文件恢复所有设置
:::
::::

::: tip 使用场景
方便服务器迁移和备份恢复，导出文件包含所有系统配置（不含用户数据）。
:::

## 个人资料

右上角头像下拉菜单可进入个人资料页面：

| 配置项 | 说明 |
|--------|------|
| **用户名** | 修改当前登录用户名 |
| **密码** | 修改登录密码 |
| **头像** | 上传自定义头像 |

# 使用统计

对应前端侧边栏 **监控 → 使用统计**，展示 Bot 使用情况的聚合报表、趋势图表和明细记录。

## 页面结构

页面从上到下分为三个区域：汇总卡片、趋势图表、明细记录表。

## 汇总卡片

顶部 4 张统计卡片，可切换「日报」/「周报」视角：

:::: card-grid
::: card title="解锁成功" icon="material-symbols:lock-open"
期间内 HDHive 解锁成功次数
:::
::: card title="搜索次数" icon="material-symbols:search"
期间内 Bot 搜索请求总数
:::
::: card title="受限次数" icon="material-symbols:block"
期间内因频率限制被拦截的请求数
:::
::: card title="活跃用户" icon="material-symbols:group"
期间内有操作记录的独立用户数
:::
::::

## Top 用户排行

汇总区域右侧展示 Top 用户排行（按解锁成功次数排序），每条记录显示用户名和次数。

## 趋势图表

使用面积图（recharts `AreaChart`）展示近 7 天的每日趋势，包含以下数据线：

- **解锁成功** — 绿色
- **搜索次数** — 蓝色
- **受限次数** — 红色

## 明细记录表

底部可分页、可搜索的记录表格：

| 列名 | 说明 |
|------|------|
| **时间** | 操作发生时间 |
| **用户** | 操作用户名 |
| **类型** | 操作类型标签 |
| **详情** | 操作具体内容 |

支持按关键词搜索，分页展示（每页 10/20/30 条可选）。

# 邀请码

对应前端侧边栏 **开服 → 邀请码**，管理 Emby 用户的邀请码（激活码），支持单个生成和批量生成。

## 页面结构

顶部标题栏包含「生成邀请码」和「批量生成」两个按钮。下方为搜索栏 + 邀请码列表表格，支持分页。

## 邀请码列表表格

| 列名 | 说明 |
|------|------|
| **邀请码** | 随机生成的邀请码字符串，点击可复制 |
| **时长** | 邀请码对应的有效天数，永久码显示「永久」标签 |
| **允许模式** | 自备 / 共享 / 两者皆可 |
| **状态** | 未使用（绿色）/ 已使用（灰色） |
| **使用者** | 已使用时显示使用者的 Emby 用户名 |
| **创建时间** | 邀请码创建时间 |
| **使用时间** | 已使用时显示使用时间 |
| **操作** | 复制 / 删除 |

## 生成邀请码弹窗

| 配置项 | 说明 |
|--------|------|
| **时长** | 下拉选择：天卡(1天) / 周卡(7天) / 月卡(30天) / 季卡(90天) / 年卡(365天) / 永久 |
| **允许模式** | 自备网盘 / 共享网盘 / 两者皆可 |
| **Emby 实例** | 选择特定 Emby 实例，或留空不限制 |

::: details 批量生成弹窗
在生成邀请码弹窗的基础上增加：

| 配置项 | 说明 |
|--------|------|
| **数量** | 一次生成的邀请码数量 |
:::

::: tip 批量生成完成后，可一键复制所有生成的邀请码到剪贴板。
:::

## 批量删除

::: warning 批量删除
表格支持多选，选中后可批量删除未使用的邀请码。已使用的邀请码不可删除。
:::

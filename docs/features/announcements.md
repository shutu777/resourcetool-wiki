# 公告管理

对应前端侧边栏 **开服 → 公告管理**，管理面向 Emby 用户的系统公告，支持推送到 Telegram Bot。

## 页面结构

顶部标题栏 + 「新增公告」按钮。下方以卡片列表展示所有公告，按创建时间倒序排列。

## 公告卡片

每张公告卡片显示：

- **标题** — 公告标题
- **内容** — 公告正文（Markdown 格式）
- **目标实例** — 该公告面向的 Emby 实例名称，或「全部实例」
- **状态标签** — 已启用（绿色）/ 已停用（灰色）
- 操作按钮：推送 / 启用或停用 / 编辑 / 删除

## 新增/编辑公告弹窗

| 配置项 | 说明 |
|--------|------|
| **标题** | 公告标题 |
| **内容** | 公告正文，支持 Markdown 格式 |
| **目标实例** | 下拉选择特定 Emby 实例，或选择「全部实例」 |
| **启用** | 开关，控制公告是否对用户可见 |

## 推送功能

::: tip 一键广播
每张公告卡片上的「推送」按钮，可将公告内容一键广播到对应 Emby Bot 的所有已绑定用户。推送完成后显示发送成功/失败的数量统计。
:::

## 状态切换

点击「启用/停用」按钮可快速切换公告的可见状态，无需打开编辑弹窗。

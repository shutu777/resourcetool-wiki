# 秒传

管理 115 网盘的定时扫描和自动秒传任务，以及查看秒传执行历史和统计数据。

## 秒传任务

对应前端侧边栏 **秒传 → 秒传任务**，管理所有秒传任务的配置。

### 页面结构

顶部标题栏 + 「添加任务」按钮。下方以卡片列表展示所有秒传任务。

### 任务卡片

每张卡片显示：

- **任务名称** + 启用状态标签（运行中 / 已停用）
- **监听目录** — 扫描的源目录路径
- **目标目录** — 秒传到的目标目录路径
- **扫描间隔** — 定时扫描的时间间隔
- 操作按钮：编辑 / 删除

### 添加/编辑任务弹窗

#### 基本配置

| 配置项 | 说明 |
|--------|------|
| **任务名称** | 自定义名称 |
| **启用** | 开关 |
| **源 115 配置** | 从哪个 115 账号的网盘扫描文件 |
| **监听目录** | 要扫描的 115 网盘源目录，通过目录选择器选择 |
| **目标 115 配置** | 秒传到哪个 115 账号 |
| **目标目录** | 秒传的目标目录，通过目录选择器选择 |

::: details 扫描设置
| 配置项 | 说明 |
|--------|------|
| **扫描间隔** | 定时扫描的时间间隔（分钟） |
| **并发数** | 同时执行秒传的最大并发数 |
| **递归扫描** | 开关。是否递归扫描子目录 |
:::

::: details 媒体过滤
| 配置项 | 说明 |
|--------|------|
| **文件类型** | 仅秒传指定类型的文件（视频 / 音频 / 字幕 / 全部） |
| **最小文件大小** | 忽略小于此大小的文件 |
| **文件名过滤** | 白名单/黑名单关键词，按文件名匹配 |
:::

### 目录选择器

::: tip 目录选择器
与 [115 助手](pan115-config.md) 中相同的目录选择器弹窗，带面包屑导航浏览 115 网盘目录。
:::

## 秒传记录

对应前端侧边栏 **秒传 → 秒传记录**，展示所有 115 秒传任务的执行历史和统计数据。

### 汇总卡片

页面顶部 4 张统计卡片：

:::: card-grid
::: card title="等待中" icon="material-symbols:hourglass-top"
当前排队中的秒传任务数
:::
::: card title="执行中" icon="material-symbols:sync"
当前正在执行的秒传任务数
:::
::: card title="成功" icon="material-symbols:check-circle"
累计秒传成功次数
:::
::: card title="失败" icon="material-symbols:error"
累计秒传失败次数
:::
::::

### 记录列表

下方为可筛选、可搜索、可分页的秒传记录表格：

| 列名 | 说明 |
|------|------|
| **时间** | 秒传执行时间 |
| **文件名** | 被秒传的文件名称 |
| **源配置** | 源 115 账号名称 |
| **目标配置** | 目标 115 账号名称 |
| **状态** | 成功（绿色）/ 失败（红色）/ 执行中（琥珀色）/ 等待中（蓝色） |
| **详情** | 点击可查看详细信息（文件路径、错误信息等） |

::: details 筛选条件
- **搜索框** — 按文件名关键词搜索
- **状态筛选** — 全部 / 等待中 / 执行中 / 成功 / 失败
- **任务筛选** — 按秒传任务名称筛选

分页展示（每页 10/20/30 条可选）。
:::

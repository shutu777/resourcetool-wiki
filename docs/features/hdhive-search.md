# 影视搜索

对应前端侧边栏 **HDHive → 影视搜索**，集成 TMDB 和 HDHive Open API，提供从搜索到解锁到转存的一站式体验。

## 页面结构

页面分为三个状态：搜索首页、搜索结果列表、影视详情页。

### 搜索首页

居中的搜索框 + 搜索类型切换（电影 / 电视剧），输入关键词后回车或点击搜索按钮触发搜索。

### 搜索结果列表

以网格卡片展示 TMDB 搜索结果，每张卡片包含：

- **海报图** — TMDB 海报（无海报显示占位图）
- **标题** — 中文名 + 原名
- **年份** — 上映年份
- **类型标签** — 电影 / 电视剧
- **评分** — TMDB 评分

点击卡片进入影视详情页。

### 影视详情页

左侧为 TMDB 媒体信息（海报、背景图、简介、评分、上映日期等），右侧为 HDHive 资源列表。

#### 资源列表

每条 HDHive 资源显示：

| 字段 | 说明 |
|------|------|
| **标题** | 资源名称 |
| **格式标签** | 分辨率、编码、音频等标签（如 4K、HEVC、Atmos） |
| **大小** | 文件总大小 |
| **积分** | 解锁所需积分 |
| **操作** | 解锁按钮 |

#### 解锁与转存

点击「解锁」后：

::: steps

1. **扣除积分**

   调用 HDHive API 扣除积分并获取资源链接

2. **自动处理**

   根据链接类型自动处理：
   - **115 分享链接** — 弹出 115 配置选择，选择后一键转存到网盘
   - **magnet / ed2k 链接** — 弹出 115 配置选择，提交离线下载任务

3. **结果通知**

   转存/离线结果通过 Toast 通知反馈

:::

## 115 配置选择弹窗

::: tip 115 配置选择
解锁后弹出转存弹窗，下拉选择要转存到哪个 115 账号，转存目录使用该 115 配置的默认转存目录。如有多个 115 账号，可选择任意一个进行转存。
:::

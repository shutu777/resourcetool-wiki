# 在线会话

对应前端侧边栏 **开服 → 在线会话**，实时展示所有 Emby 实例的活跃会话，支持按实例切换查看。

## 页面结构

顶部标题栏 + Emby 实例选择下拉 + 自动刷新开关。下方按三个分组展示会话：

:::: card-grid
::: card title="正在播放" icon="material-symbols:play-arrow"
当前有播放活动的会话，绿色高亮显示
:::
::: card title="在线" icon="material-symbols:wifi"
已连接但未在播放的会话
:::
::: card title="离线" icon="material-symbols:wifi-off"
最近连接过但已断开的会话
:::
::::

每组显示会话数量徽章。

## 会话卡片

每个会话以卡片展示，包含以下信息：

| 字段 | 说明 |
|------|------|
| **用户名** | Emby 用户名 |
| **客户端** | 播放器名称和版本（如 Infuse 7.8） |
| **设备名** | 设备标识（如 iPhone 15 Pro） |
| **IP 地址** | 客户端 IP |
| **播放内容** | 当前正在播放的媒体名称 |
| **播放进度** | 进度条 + 已播放时间 / 总时长 |
| **播放方式** | 直接播放 / 转码 标签 |
| **码率** | 播放码率 |

## 播放进度

正在播放的会话显示实时进度条，进度条下方标注已播放时间和总时长。进度数据随自动刷新更新。

## 终止会话

::: danger 终止会话
每个正在播放的会话卡片右上角有「终止」按钮，点击后确认即可强制断开该会话的播放。此操作会立即中断用户的播放。
:::

## 自动刷新

::: tip 自动刷新
页面默认开启自动轮询（约 5 秒一次），可通过顶部开关关闭。手动刷新按钮始终可用。
:::
